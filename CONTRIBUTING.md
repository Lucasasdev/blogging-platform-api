### Pre-requirements
- [Node LTS](https://nodejs.org/en)
- [NPM 10.5 >=](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

1. Clone the project
```
https://github.com/Lucasasdev/web-api.git
```
2. Install the dependences as following:
```
npm install
```
3. Run de project
```
npm run dev
```
